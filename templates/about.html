About

We began the project as a way to practice data representation and other front end work like templating with Jinja as well as back end work like Python Flask. We were tasked to choose a data set, and after some peer feedback, I landed on energy usage within the US, leading us here.

1. I retrieved data from The US Energy Information Administration.

2. The data was in xlsx format so I converted it to csv.

3. I reorganized and edited rows and columns to simplify the python process.

4. I used a python script to parse the data into python dictionaries.

4. I assumed all negative values were zero because I'm working solely with solely positive energy production.

Feedback

For feedback, I was told to include more color, inprove labeling, increase clarity, change the background color, and produce a brief written description of data representations.

Takeaways

My takeaways are to improve on the feedback as well as to be more meticulous and precise with my data representations.

All data from The US Energy Information Administration.
